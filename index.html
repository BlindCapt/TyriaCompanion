Configuration de la Clé API

Pour charger vos personnages, votre banque et le suivi de vos succès, saisissez une clé API Guild Wars 2.

Comment obtenir une clé API ?

  1. Connectez-vous sur account.arena.net.
  2. Allez dans l'onglet Applications puis Créer une clé.
  3. Cochez les permissions : account, characters, inventories, wallet, progression et unlocks.
  4. Copiez la clé générée ci-dessous.
